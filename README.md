# 介紹  
簡易的記帳應用程式


# 功能  
可以登入後可以新增修改刪除記帳項目, 統計總金額, 項目可區分為五大分類

# 使用  
安裝node.js and npm  
clone 專案  
npm install  
npm run dev  
瀏覽器輸入 http://localhost:3000  
npm run seed
